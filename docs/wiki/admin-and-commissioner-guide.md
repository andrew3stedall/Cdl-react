# Admin and Commissioner Guide

Commissioners configure seasons, manage invitations, start drafts, oversee approvals, and handle corrections.

## Roles

- Commissioner
- Vice commissioner
- Manager
- Spectator

## Approvals

Transfers and loans require approval after both parties agree. If the commissioner is involved, the vice commissioner must approve.

## Corrections

Corrections require a reason and should create audit records. They should not silently overwrite competitive history.

Correction examples:

- fixture result correction
- lineup correction
- squad correction
- draft pick correction
- free agency draw correction
- rule correction

## Approval queue

Commissioners and vice commissioners should have a dedicated approval queue showing pending items, affected teams, players, rule checks, and conflict-of-interest status.
