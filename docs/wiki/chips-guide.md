# Chips Guide

Each team can use one chip per gameweek. Chips lock at the FPL deadline with the lineup.

## Chips

| Chip | Effect |
|---|---|
| Triple Captain | Captain scores 3x instead of 2x. |
| Dual Captain | Captain and vice captain both score 2x. |
| Auto Captain | Highest scorer from the scoring lineup gets captain multiplier. |
| Bench Boost | Bench players score; reserves excluded. |
| Best XI | Best 11 from starters plus bench score, ignoring position constraints. |

## Chip impact

Fixture detail should show whether the chip was useful:

- points added
- whether it was wasted
- whether it changed the fixture result
- whether it changed league points
