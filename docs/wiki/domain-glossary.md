# Domain Glossary

## Core terms

| Term | Meaning |
|---|---|
| League | Persistent CDL competition container across seasons. |
| League season | One season of a league. |
| Team | Persistent team identity within a league. |
| Season team | A team's participation in one season, including display name and manager. |
| Squad | Players controlled by a season team. |
| Matchday team | Players selected for a gameweek lineup. |
| Scoring lineup | Players who count after substitutions and chips. |
| Transfer | Player movement between teams or through approved squad changes. |
| Free transfer | Player acquired through a free agency draw. |
| Loan | Temporary player movement with scheduled return. |
| Temporary right | Short-lived right to add a draw-won player before the FPL deadline. |
| Draft | Live sequential player allocation before season. |
| Free agency draw | Automated preference-based allocation in configured gameweeks. |
| Chip | One-use scoring modifier. |
| Fixture | CDL head-to-head match. |
| Knockout tie | One or more fixture legs that determine advancement. |
| Snapshot | Stored calculated state at a point in time. |

## Preferred language

Use **squad**, not roster.
Use **transfer**, not transaction.
Use **free transfer**, not free-agent move.
